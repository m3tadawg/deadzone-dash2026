# DeadZone Dash — Project Specification

## Core Loop
Spawn → Survive → Loot → Extract or Die

## Architecture
- Server authoritative
- Client = rendering + prediction
- Modular systems only

## Systems
- Combat
- AI
- Spawn
- Weather
- Events

## Design Principles
- Data-driven (JSON)
- No monolithic files
- ECS or manager-based separation
